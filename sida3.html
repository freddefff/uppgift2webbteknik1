Vad är en regel i CSS och vad består den av?

Regel i CSS - används för att definiera styling för ett element i HTML. Dessa består av selektor och regeluppsättning.

Vad är skillnaden mellan id och class? När bör man använda id respektive class?

ID - En unik identifierare för ett specifikt element. Klass - Dessa kan exmepelvis användas på flera olika element för att kunna dela samma stil. Använd ID - används mer för unika och specifika element. Använd klass - används för att kunna återanvända samma stil på flera olika element.
